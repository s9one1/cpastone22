## 03월 23일
### 개발일지
> 글쓰기 화면 만들기 과정
1. 글쓰기 화면의 유저 시나리오를 작성한다.
```
유튜브에서 영어 영상을 보고 글을 작성하려는 김민정
1. 글쓰기 아이콘을 누른다.
2. 자신이 본 영상의 링크를 복사해와서 붙여넣는다.
3. 영상을 보고 무엇을 배웠는지, 얼마나 성장했느지 등의 기록과 감상을 써넣는다.
4. 캘린더에 공부 기록을 올리기 위해 피드용과 캘린더용으로 올리기 버튼을 누른다.
```
